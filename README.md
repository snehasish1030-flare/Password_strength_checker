# High-Tech Password Strength Checker

A **cyberpunk-style**, **fully client-side** password strength analyzer that runs **100% in the browser** — no server, no backend, no Python needed.

Built with HTML, CSS, JavaScript, and [Particles.js](https://github.com/VincentGarreau/particles.js/) for a futuristic look.

---

## Features

- **Visible password input** – see exactly what you type  
- **Real-time entropy calculation** (in bits)  
- **Accurate strength scoring** (Weak, Moderate, Strong, Very Strong)  
- **Common password detection** (e.g., "password", "123456")  
- **High-tech UI**:  
  - Neon glow effects  
  - Animated particle network background  
  - Subtle CRT scanline overlay  
  - Smooth bounce-in result animation  
- **Zero dependencies** (except CDN for Particles.js)  
- **Deploy instantly on GitHub Pages**
